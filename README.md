🟢 Landing Page - WordPress (Prueba Técnica APROS)

Este proyecto corresponde a la implementación de una landing page desarrollada en WordPress como parte de una prueba técnica para el puesto de Practicante de Desarrollo / Soporte Web Junior.

📌 Descripción

Se desarrolló una landing page utilizando el tema Twenty Twenty-Five, aplicando personalización mediante HTML, CSS y JavaScript dentro del entorno de WordPress.

El objetivo fue construir una interfaz funcional, responsive y con interacción básica del usuario.

✨ Funcionalidades implementadas
Diseño responsive adaptado a dispositivos móviles y desktop
Sección de servicios con tarjetas dinámicas
Filtro de servicios activos utilizando JavaScript
Ordenamiento de servicios por precio (menor a mayor)
Interacción con botones mediante JavaScript puro (DOM manipulation)
🧠 Tecnologías utilizadas
WordPress (local)
HTML5
CSS3
JavaScript (Vanilla JS)
Tema: Twenty Twenty-Five
📂 Estructura de la implementación
wordpress-landing/
│
├── landing.html
├── styles.css
├── script.js


🧠 Decisiones técnicas
Se utilizó JavaScript puro para mantener simplicidad y control del DOM.
Se evitó el uso de librerías externas para demostrar lógica manual.
Se priorizó un diseño limpio y funcional antes que complejidad visual.
Se trabajó directamente sobre WordPress local para simular entorno real.
🚀 Posibles mejoras futuras
Separar CSS y JS en archivos optimizados y minificados
Mejorar accesibilidad (ARIA labels, navegación teclado)
Agregar animaciones suaves en transiciones
Implementar buscador de servicios
Optimización de rendimiento (lazy load, imágenes WebP)
Mejoras en UX mobile-first
📌 Nota

La instalación completa de WordPress no se incluye en el repositorio debido a su tamaño y carácter local. En su lugar, se documenta la implementación mediante código fuente y capturas del resultado final.
